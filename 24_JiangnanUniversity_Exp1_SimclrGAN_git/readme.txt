文件夹：
    a_result存储生成对抗网络生成的基准图像
    checkpoints文件夹存储四种环境的生成器、判别器的模型，simclr网络模型以及分类器模型
    d_result存储最终生成的迷彩图像
    data_{}文件夹存储四种环境的训练图像，可以通过数据增强增加至九百张
    modles文件夹存储生成对抗网络结构
hdf5文件：
    train_hdf5_{}存放经过整理的某种环境模式下的训练集。hdf5可以加快数据的存取速度。
txt文件：
    text_{}.txt文件存放某环境训练集的对应语义
    text_word2vec{}.txt存放某环境的词袋模型训练结果
py文件：
    a开头的py文件是针对模型的第一部分————基准图像生成模型的数据集整理、训练、验证的一系列操作
    b开头的py文件是针对模型的第二部分————SimCLR网络模型的数据集整理、训练的一系列操作
    c开头的py文件是针对模型的第三部分————SimCLRClassifier多分类器训练的一系列操作
    d开头的py文件负责运行。请运行d2_generate_camouflage.py文件生成特定环境的迷彩图像。
    model_to_png类的img_num参数调控一次性生成的迷彩图像数量，model.main()负责生成。
    请输入合法数值，格式为：{1}:{2} 其中，{1}输入环境模式，包括[forest、desert、rivers、snowed]四种，不要输错字母
    {2}是输入的针对图像的语义描述，请使用text_{}.txt内包含的词语。
    生成的图像保存至d_result文件夹中。
